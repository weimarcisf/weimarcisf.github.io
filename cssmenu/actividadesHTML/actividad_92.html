El elemento <div> permite agrupar un grupo de elementos juntos en un cuadro de nivel de bloque. Por ejemplo, puede crear un elemeto <div> para que contenga todos los elementos del encabezado del sitio (el logotipo y la navegación),o puede crear un elemento <div> para que contenga información en bloques.

En un navegador el contenido del elemento <div> comenzará en una nueva linea, pero aparte de esto no hará ninguna diferencia en la presentación de la pagina.Sin embargo, el uso de los atributos ID o Class en el elemento <div> significa que puede crear reglas de estilo CSS para indicar cuánto espacio debe ocupar elelemento <div> en la pantalla y cambiar la apariencia de todos los elementos contenidos en él.También puede faicilitar el seguimeiento del código si ha utilizado elementos <div> para contener cada sección de la página.

Dado que pude haber varios otros elementos dentor de un elemento <div>, puede ser útil agregar un comenatario despues de la etiqueta </div> de cierre. Esto le permite ver claramente a que etiqueta de apertura se supone que corresponde.
